Trust-Region Methods

  1. Given x(k)

  2. Replace f by (e.g 2nd order) approximation f^

  3. Solve x^=argminx∈Dkf^(x) for a given thrust region Dk={x∈Rn∣∥x−x(k)∥p≤δ}

  4. Test improvement ρ=actual improvementpredicted improvement=f(xk)−f(x^)f(xk)−f^(x^)

  5. If ρ>ρmin, set x(k+1)=x^, else decrease thrust region radius δ←σδ
