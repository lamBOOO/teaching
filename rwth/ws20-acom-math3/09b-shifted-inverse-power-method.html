Define the (Shifted) Inverse Power Method Algorithm

  1. Given A∈Rn and eigenvalue shift μ

  2. Choose start vector x0∈R∖{0}

  3. While k<kmax∧error>tol do

    1. Solve: (A−μI)xk+1=xk (this is like xk+1=(A−μI)−1xk)

    2. Normalize: xk+1↦xk+1/||xk+1||2

    3. Update eigenvalue estimate (with initial matrix A, not A−1): λk+1=ρA(xk+1)

  4. Return estimated eigenpair (λk,xk)
