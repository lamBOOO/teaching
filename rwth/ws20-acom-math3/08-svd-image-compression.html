Image Compression Using SVD

We consider an grayscale image A∈Rn×n with entries Aij∈[0,1] representing the gray intensity.

Singular Value Decomposition

A square matrix A∈Rn×n can be written as SVD, defined as:

A=UΣVT=∑i=1nuiσiviT=u1σ1v1T+⋯+urσrvrT

Compressed Image

We construct the compressed image A~∈Rn×n as rank r=13 approimxation, defined as:

A~=∑i=1ruiσiviT=u1σ1v1T+⋯+urσrvrT

with rank r.

Storage Requirement of Compressed Matrix

Instead of storing n2 matrix entires, we could now only store the r-times the summation tuple {ui,σi,viT} which leads to a size

size(A~)=r(n+1+n)=r(2n+1)≪n2=size(A) for r≪n
