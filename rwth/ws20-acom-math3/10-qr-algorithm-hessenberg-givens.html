QR Algorithm Native

We use Julia's standard qr() function and implement:

  1. Given A∈Rn×n

  2. Initialize Q(m+1)=I

  3. For k=1,…,m:

    1. Calculate QR-Decomposition: Ak=QkRk

    2. Update: Ak+1=RkQk

  4. Return diagonal entries of Am and Q(m)=Qm⋯Q0Q1

Improve QR Algorithm with Upper Hessenberg Matrix Preconditioning

  • Idea: QR decomposition needs O(n3), QR for Hessenberg matrices is easier done in O(n2). Linear complexity is even possible if A is symmetric. In this case, the QR decomposition only needs O(n) Givens rotations with constant effort.

  • Therefore transform the matix A to upper Hessenberg form with similarity transforms in O(n3) (also cubic, but only needs to be done once) and use this matrix for the QR algorithm.

Upper Hessenberg Shape

H=(h11h12h13⋯h1nh21h22h23⋯h2n0h32h33⋯h3n⋮⋱⋱⋱⋮0⋯0hnn−1hnn)

Algorithm [1]:

  1. Given: A∈Rn×n

  2. For k=1…n−2 do:

    1. [v,β]←house(A(k+1:n,k))

    2. A(k+1:n,k:n)←(I−βvvT)A(k+1:n,k:n)

    3. A(1:n,k+1:n)←A(1:n,k+1:n)(I−βvvT)

with Householder reflection vector v and weight β=2/(vTv).

[1]: https://www.tu-chemnitz.de/mathematik/numa/lehre/nla-2015/Folien/nla-kapitel6.pdf

QR Algorithm for symmetric Hessenberg matrices

Symmertric hessenberg matrices are tridiagonal (only diag plus uppe and lower sub-diagonal). For the QR decomposition, we only have to make the lower sub diagonal entries to zero to obtain the upper right triangular matrix. This can be done by using Givens roations:

Givens Rotations [1]

Given a matrix A, we can make to entry Aij to zero with Anew=G(A,i,j)A where

G(A,i,j)=[1⋯0⋯0⋯0⋮⋱⋮⋮⋮0⋯c⋯−s⋯0⋮⋮⋱⋮⋮0⋯s⋯c⋯0⋮⋮⋮⋱⋮0⋯0⋯0⋯1],

with

  • r=Ajj2+Aij2

  • s=−Aij/r

  • c=Ajj/r

[1]: https://en.wikipedia.org/wiki/Givens_rotation
